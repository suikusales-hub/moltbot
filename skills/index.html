Moltbot Skills & Capabilities

Quick Summary

Skills are the functional modules that give Moltbot its powers. Without skills, Moltbot is just a conversationalist; with skills, it becomes an agent capable of browsing the web, generating images, managing calendars, and writing code. This page reviews the extensive library of skills available through MoltHub and ClawdHub. We categorize skills into core utilities, productivity tools, and entertainment modules. You will learn how to install new skills, configure their parameters, and even develop your own custom skills using the Moltbot SDK.

Frequently Asked Questions

How do I write my own skill?
Skills are written in TypeScript/JavaScript. You define the function schema for the LLM and the implementation logic. Check the developer docs for the SDK.
Are skills safe to use?
Official skills on MoltHub are vetted, but you should always review the code of 3rd party skills before installing them on your server.
