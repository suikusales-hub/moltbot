Frequently Asked Questions

Can Moltbot handle sensitive data?
Since Moltbot is self-hosted, you retain full control over your data. However, be mindful of what data you send to third-party LLM providers.
Is it suitable for high-traffic environments?
Yes, Moltbot can scale horizontally. Using Redis for session management allows you to run multiple instances behind a load balancer.
