04. Tools & Security Risks

The Agent Runtime decides actions, not just words. If you ask to "Delete emails," the agent invokes the Email Tool. This power comes with significant responsibility.

Exposed Gateway

By default, the Gateway listens on localhost. Exposing this port publicly allows anyone to control your agent.

Untrusted Senders

Without allow-listing, any user on a connected platform (e.g., a random Telegram user) could trigger commands.

Tool Scope

Tools run with the host's permissions. A `bash` tool has the same power as the user running the process.
