01. Prerequisites

  • > OS: macOS 14+ or Ubuntu 22.04+ (Windows via WSL2)
  • > Runtime: Node.js v20.0.0 or higher
  • > Docker: Optional, but recommended for sandboxed skills
  • > LLM API Key: OpenAI, Anthropic, or local Ollama

02. The One-Line Install

Moltbot provides a unified installer that handles dependencies and service creation. Open your terminal and run:

curl -fsSL https://get.moltbot.com/install.sh | bash
bash

*Note: This script will prompt you for sudo access to install the system service.

03. Configuration & Onboarding

Once installed, initialize your bot identity. This interactive wizard will guide you through selecting your LLM provider and setting up your "System Prompt" (personality).

moltbot onboard

The configuration is stored in ~/.moltbot/config.json (legacy paths may use ~/.clawdbot). You can edit this file manually later for advanced settings like context window size or temperature.

04. Connecting Channels

The real magic happens when you connect a messaging platform. Moltbot acts as a "client" on these networks.

WhatsApp

Scan a QR code to link Moltbot as a linked device.

moltbot channel add whatsapp

Telegram

Use a Bot Token from @BotFather.

moltbot channel add telegram --token YOUR_TOKEN

05. Setting Up Heartbeats

"Heartbeats" allow Moltbot to wake up autonomously and check for tasks, even if you haven't messaged it. Create a file at ~/moltbot/HEARTBEAT.md: 

# Routine Check
- [ ] Check unread emails in Gmail
- [ ] Verify disk space usage on /var
- [ ] If CPU load > 80%, send me a warning on Telegram
- [ ] Summarize yesterday's git commits

Moltbot will process this checklist every hour (configurable) and execute necessary actions.
