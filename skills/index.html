Moltbot Skills & Capabilities

Quick Summary

Skills are the functional modules that give Moltbot its powers. Without skills, Moltbot is just a conversationalist; with skills, it becomes an agent capable of browsing the web, generating images, managing calendars, and writing code. This page reviews the extensive library of skills available through MoltHub (formerly ClawdHub). We categorize skills into core utilities, productivity tools, and entertainment modules. You will learn how to install new skills, configure their parameters, and even develop your own custom skills using the Moltbot SDK.

01 // What are Skills?

In the context of LLM agents, a "Skill" (often called a "Tool" or "Function") is a structured piece of code that the AI can execute. When you ask Moltbot to "Find the cheapest flight to Tokyo," the LLM doesn't guess; it recognizes the intent and calls the flight_search skill with parameters like { destination: "Tokyo", sort: "price" }.

Moltbot supports the OpenAI Function Calling standard, making it compatible with thousands of existing definitions.

05 // Installing Skills from MoltHub

MoltHub is the community registry for skills. You can install skills using the CLI directly inside your Moltbot container:

moltbot skill install github-ops
moltbot skill install linear-integration

/// FAQ

01 // How do I write my own skill?
Skills are written in TypeScript/JavaScript. You simply define the Zod schema for the arguments and an asynchronous function for the logic. Our SDK handles the JSON-RPC communication with the LLM.
02 // Are skills safe to use?
Official skills on MoltHub (marked with a blue badge) are audited by the core team. For third-party skills, we recommend reviewing the source code, as skills run with the same privileges as the Moltbot process itself.
