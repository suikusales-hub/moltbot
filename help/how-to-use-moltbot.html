01. Environment Preparation

Before initializing the Moltbot core, ensure your environment is primed. As a local runtime, Moltbot relies on the underlying capabilities of your operating system.

Component Requirement Notes
Operating System macOS, Linux, or WSL2 Windows users must use WSL2 (Ubuntu). Native Windows support is experimental.
Runtime Node.js v22+ Critical for modern ESM support and fetch APIs.
Memory 4GB RAM (Min) 8GB+ recommended if running local LLMs like LLaMA 3.
API Access LLM Provider Key OpenAI, Anthropic, or a local Ollama endpoint.

Windows Users: Do not attempt to run Moltbot in PowerShell directly. The file system watchers and signal handling require a POSIX-compliant environment. Install Ubuntu via WSL2 from the Microsoft Store before proceeding.

02. Installation Protocol

We provide a streamlined installation script that handles dependencies, directory structure, and service registration. Open your terminal and execute the following command: 

curl -fsSL https://get.moltbot.com/install.sh | bash

The Onboarding Wizard

Once the binaries are installed, you need to initialize your bot's identity. Run the onboarding command to start the interactive configuration wizard:

moltbot onboard

This wizard will guide you through:

  • Selecting your primary LLM Provider (e.g., GPT-4o, Claude 3.5 Sonnet).
  • Defining the System Prompt (your bot's personality and core directives).
  • Setting up Storage Persistence (Redis or JSON file system).

For a step-by-step walkthrough of this process, refer to our Zero to Agent Tutorial.

05. Cron & Autonomy

Moltbot isn't limited to reactive tasks. You can configure Proactive Routines using standard Cron syntax. This allows your agent to wake up at specific intervals and perform health checks, generate reports, or clean up temporary files without any human input.

Example: The Morning Briefing

Configured in ~/.moltbot/routines.json

{
  "schedule": "0 8 * * *",
  "task": "Check my calendar for the day, summarize unread emails from 'Important' label, and check server CPU usage. Send a summary to Telegram."
}
